AutoCAT

This project was created to automate a common task performed by the Boutsy team. Given the simple and repetitive nature of creating vendor categories on our site, it seemed fitting to have the task automated from the browser.

The following skills and libraries were used when creating this project:

  • ◼️ Gooey - A Simple GUI Builder
  • ◼️ Selenium Python API
  • ◼️ XPath - HTML Locators
  • ◼️ Python Decorators and Type Hinting
  • ◼️ Python Classes, Methods, and Error Handling

The AutoCAT started as a simple script that copy and pasted text from one HTML form element to another, and has developed into a fully functional GUI program. The majority of the code implements Selenium to target HTML elements using XPaths, and then pastes / formats the appropiate text into a defined field. Given just the vendor's email address, the program will automatically create the vendor's category through the browser and the site's UI in an average execution time of 35 seconds. It launches tabs and fills out fields as soon as the page loads, then moves to the next page as soon as the altered data is saved. Have a look at how this was pulled together here.

UPDATE: I have added an extra feature to the program that collects and saves the vendor's account information to a Mongo database!
