AutoCAT

This project was created to automate a common task performed by the Boutsy team. Given the simple and repetitive nature of creating vendor categories on our site, it seemed fitting to have the task automated from the browser.

The following skills and libraries were used when creating this project:

  • ◼️ Gooey - A Simple GUI Builder
  • ◼️ Selenium Python API
  • ◼️ XPath - HTML Locators
  • ◼️ Python Decorators and Type Hinting
  • ◼️ Python Classes, Methods, and Error Handling

The AutoCAT started as a simple script that copy and pasted text from one HTML form element to another, and has developed into a fully functional GUI program. The majority of the code implements Selenium to target HTML elements using XPaths, and pastes / formats the appropiate text into the defined field. Given just the vendor's email address, the program will automatically create the vendor's category in an average execution time of 35 seconds. Have a look at how this was pulled off here.
