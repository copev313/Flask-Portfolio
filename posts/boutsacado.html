Boutsacado

In this project I built an eCommerce platform from scratch using React, Redux, Django & Postgres.

Some more of the tools that were utilized are listed below:

  • ◼️ React-Bootstrap
  • ◼️ React Router
  • ◼️ React Redux
  • ◼️ Bootstrap / Bootswatch Themes
  • ◼️ Local Storage API & Axios
  • ◼️ Django REST Framework
  • ◼️ AWS S3 Buckets & RDS for PostgreSQL
  • ◼️ Heroku

The goal of this project was to create a full featured eCommerce platform based around a Django backend and React frontend. It incorporates proper product, user, and order management. There is a complete checkout process including Paypal intregration and order tracking. The site implements user profiles to manage account information and placed orders. Products can be searched for, listed with pagination, and reviewed by registered users. This application also includes state management using Redux and JWT authentication. Take a look at the live site here: www.boutsacado.com.
