Slash Commands Google Chat Bot

This project was built to bring back the power of slash commands in Google Chat. It is developed and deployed from Google Apps Script using Google Cloud.

Here are some of the APIs used to generate the bots responses:

  • ◼️ NASA API
  • ◼️ XKCD API
  • ◼️ Cuttly API
  • ◼️ Jokes API
  • ◼️ Dog API
